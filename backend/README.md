# Backend

This folder contains backend source files for SmartTimetable.

Current structure:
- `app.py`: Flask backend entry (copied from root backend app)
- `models/`: database operations
- `scheduler/`: timetable generation engine
- `config/`: backend settings
- `routes/`, `services/`, `utils/`, `schemas/`: modular extension points

Run from project root:

```bash
python app.py
```

You can also run:

```bash
python backend/app.py
```
